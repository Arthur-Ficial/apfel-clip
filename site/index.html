apfel-clip

AI clipboard actions for macOS.
Copy text. Press ⌘⇧V. Pick an action. Done.

Download

↓ Download free (macOS, Apple Silicon)

or install with Homebrew

brew install --cask arthur-ficial/tap/apfel-clip
1
Copy anything Text, code, an error message — whatever's on your clipboard.
2
Press ⌘⇧V apfel-clip appears in your menu bar, ready to go.
3
Pick an action Fix grammar, translate, explain, reformat — one tap.
4
Paste the result Copied to clipboard automatically. Nothing left your Mac.
✅ No setup needed. Powered by apfel — the on-device AI engine — which is bundled inside the app. No API keys, no cloud, works offline.
